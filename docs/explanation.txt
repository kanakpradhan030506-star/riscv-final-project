## Project Overview
This project implements a 32-bit Single-Cycle RISC-V (RV32I) Processor using Verilog HDL. The processor executes one instruction per clock cycle and follows a modular RTL design suitable for FPGA implementation and ASIC design flow.
The processor supports arithmetic, logical, shift, comparison, memory access, branch, jump, and upper-immediate instructions from the RV32I ISA.

### Implemented Features
- 32-bit Single-Cycle RV32I Processor
- Program Counter (PC)
- Instruction Memory
- Register File
- Immediate Generator
- Control Unit
- ALU Control & 32-bit ALU
- Data Memory
- Branch and Jump Logic
- Functional Verification using EDA Playground

### Supported Instructions
ADD, SUB, ADDI, AND, OR, XOR, SLL, SRL, SRA, SLT, SLTU, LW, SW, LUI, AUIPC, BEQ, BNE, BLT, BGE, BLTU, BGEU, JAL and JALR.

This project forms a foundation for future enhancements such as pipelining, hazard detection, forwarding, branch prediction, FPGA implementation, and RTL-to-GDSII flow.
